Landing page: full page

This is the most simple use case of a Piano inline template: the whole page is Piano template. It contains everything including top bar, side bar, and/or footer, and everything else needed.

All you need to do is to prepare an empty <div> for the Piano template on a completely empty page (empty HTML body). Also some CSS should be added to your page which would remove default gaps on the edges:

        html,
        body {
          margin: 0;
          padding: 0;
        }

Demo To return from the demo page, please use the browser Back button. We were trying to make the demo as close to real life as possible, and did not provide any link to go back intentionally.

Setting up a template

There're 2 ways to set up a landing page: on your end or in Composer. More information on this you will find in our documentation, in the links just mentioned. Please pay attention to an inline type of templates.

Specifying a template size

Piano inline templates (which a full landing page is) simply take 100% of a parent container width. By the parent container we mean that <div> you specify on your end for Piano template to show up. Regarding its height, the browser counts it automatically according to the template contents. So nothing needs to be defined. Oppositely, please, let Piano template be parsed freely — things like setting up a fixed height would prevent it from displaying correctly.
